User Guide: Input Form for RNA Sequence Analysis

Step 1: Enter Your Email Address

In the first field, you should enter your email address. This is important so that results can be sent to you after the form is processed.

Step 2: Provide the Target RNA Sequence

Enter the RNA sequence you wish to analyze. The sequence should only contain valid nucleotide characters, which include A, C, G, U (uppercase), and their lowercase equivalents (a, c, g, u).

Note: If you enter characters outside of these (e.g., numbers or special symbols), the form will return an error indicating the sequence is invalid.

Step 3: Select the Organism Type

Choose the organism type from the dropdown list. Options include *E.coli* (prokaryote), *Saccharomyces cerevisiae* (yeast), and *Homo sapiens* (human).

Step 4: Choose an Optional Reporter Gene

If you wish to include an optional reporter gene, select one from the dropdown menu. The available options are 'None' (if no reporter gene is needed), 'GFP' (Green Fluorescent Protein), or 'mCherry' (a red fluorescent protein).

Note: If you provide a reporter gene sequence manually, you cannot select one from this list, and vice versa. You can only choose one option.

Step 5: Specify if You Have a Known Trigger Sequence

Indicate whether you have a known trigger sequence. If so, check the box labeled "Got a known trigger?" and enter the trigger sequence in the provided input field. Make sure the sequence length matches the organism type you selected:

  • For E.coli, the trigger must be exactly 23 nucleotides long.
  • For Saccharomyces cerevisiae or Homo sapiens, the trigger must be 30 nucleotides long.

Note: If you provide a trigger sequence that does not match the required length, you will receive an error message.

Step 6: Enter the Reporter Gene Sequence

If you did not select a reporter from the dropdown menu, you can provide a custom reporter gene sequence here. Ensure the sequence contains valid nucleotide characters.

Note: You cannot input both a reporter gene sequence and select an optional reporter at the same time. You must choose one or the other.

Step 7: Upload Your Transcript File

Upload the file containing your transcript sequences. The accepted file formats include: .fasta, .fa, .fastq, .gb, .gbk, .embl, .phy, .phylip, .aln, .nex, .stockholm, .tab, .qual, .abi, .sff, .xml, or .ig.

Note: If you upload a file in an unsupported format, the form will return an error indicating the file format is invalid.

Step 8: Submit the Form

Once all the required fields are completed, click the "Submit" button to submit the form for processing. You will receive the results via email based on the input you provided.
